# 📋 Task Tracker CLI

A simple command-line interface (CLI) project to track your tasks — including adding, updating, marking, and listing tasks.

## 🚀 Features

- Add tasks
- Update task titles
- Delete tasks
- Mark tasks as `in-progress`, `done`, or `not-done`
- List all tasks or filter by status
